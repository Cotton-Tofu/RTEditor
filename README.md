# RTEditor

This software is for editing rich text files and plain text files. Designed with accessibility in mind, it offers a Word-like user experience while being easy to use on touch-screen devices.

# Function

**Ribbon**

This software uses a ribbon interface, providing a user experience similar to that of Word or WordPad.

<img width="262" height="68" alt="image" src="https://github.com/user-attachments/assets/34b90bd5-2681-4895-8f3f-7cc3e38df647" />　<img width="300" height="300" alt="image" src="https://github.com/user-attachments/assets/a4f7a4f2-a05d-42c5-9478-8c1e1c87597b" />



**Mini Toolbar and Radial Menu**

You can display a mini toolbar when selecting text with the mouse. Alternatively, you can choose to display a radial menu instead of the mini toolbar in the settings. (You can also hide the menu when selecting text in the settings, but the context menu accessed via right-click will still be displayed.)

**Multi-Document-Interface(MDI)**

This software employs a Multi-Document Interface (MDI), allowing you to display multiple documents simultaneously.
